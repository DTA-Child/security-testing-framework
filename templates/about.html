About Security Testing Framework

Comprehensive web security analysis with industry-standard tools

Overview

The Security Testing Framework is a comprehensive web security scanner that integrates multiple industry-standard tools to provide thorough vulnerability assessments. It combines the power of OWASP ZAP, Nuclei, and Nikto to deliver real, actionable security insights without any fake or demo data.

Key Features
  • Real Vulnerability Detection
  • Multiple Scanner Integration
  • OWASP Top 10 2024 Compliance
  • REST API
  • Web Interface
  • Real-time Progress
  • Docker Support
  • Professional Reports
Integrated Security Scanners
OWASP ZAP

HTTP security analysis, SSL/TLS configuration checking, and information disclosure detection.

Nuclei

Template-based vulnerability scanning with community-driven detection templates and CVE identification.

Nikto

Web server vulnerability scanning, directory enumeration, and server fingerprinting.

Quick Start

Web Interface:

  1. Enter target URL on the home page
  2. Select desired scanners (or use all)
  3. Click "Start Security Scan"
  4. Monitor real-time progress
  5. Review detailed results

API Usage:

curl -X POST "http://localhost:8000/api/scan" \
  -H "Content-Type: application/json" \
  -d '{"target_url": "https://example.com"}'
